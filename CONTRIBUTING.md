# Contributing to The 7 Framework

## Most Valuable Contribution: Interventions

The framework defines WHERE people are. What's missing: **what MOVES each characteristic**.

If you've found something that shifts a characteristic — personally, clinically, or in research — document it.

## How to Add an Intervention

1. Fork this repo
2. Edit the relevant file in `/interventions/` (e.g., `guard.md` for GUARD interventions)
3. Add your intervention using this format:
4. Submit a Pull Request

### Intervention Format

```markdown
### [Intervention Name]

**Moves**: [CHARACTERISTIC] from [position] toward [position]
**Type**: [therapy/medication/lifestyle/somatic/cognitive/behavioral/other]
**Evidence**: [research/clinical/anecdotal/personal]
**Source**: [citation or "personal experience"]

**What it is**:
Brief description.

**How it works**:
Mechanism of action (if known).

**Notes**:
Any caveats, who it works for, contraindications.
```

### Example

```markdown
### Box Breathing (4-4-4-4)

**Moves**: GUARD from up toward tuned
**Type**: somatic
**Evidence**: research
**Source**: Ma et al. 2017, "The Effect of Diaphragmatic Breathing..."

**What it is**:
Inhale 4 counts, hold 4, exhale 4, hold 4. Repeat.

**How it works**:
Activates parasympathetic nervous system via vagal tone.
Slows heart rate, reduces cortisol.

**Notes**:
Works best for acute anxiety spikes. Less effective for chronic baseline elevation.
```

## Other Contributions

### Research
- Add validation studies to `/research/`
- Include citations for intervention evidence
- Propose measurement approaches

### Translations
- Translate framework files to `/translations/[language-code]/`
- Note: Body metaphors may need cultural adaptation

### Assessment Tools
- Improve questions in `/assessment/`
- Build digital tools (apps, web interfaces)
- Create visualizations

### Framework Refinements
- Open an Issue first to discuss changes to The 7 definitions
- Propose additional characteristics (requires strong justification)
- Suggest better spectrum language

## Quality Standards

### For Interventions
- Be specific about direction (which way does it move?)
- Note evidence level honestly
- Include who it works for / doesn't work for
- Personal experience is valid — just label it clearly

### For Research
- Include full citations
- Note sample sizes and limitations
- Distinguish correlation from causation

## What We're NOT Looking For

- Marketing for specific products/services
- Unsubstantiated claims
- Content that pathologizes normal human variation
- Anything that undermines person-first framing

## Questions?

Open an Issue to discuss before submitting large changes.
