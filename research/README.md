# Research

References, validation studies, and evidence supporting The 7 Framework.

## Primary Sources (Legacy Systems)

The framework was developed in response to limitations of:

- **DSM-5-TR** (APA, 2022) — Categorical diagnostic system
- **ICD-11** (WHO, 2022) — International classification
- **RDoC** (NIMH) — Research domain criteria
- **HiTOP** — Hierarchical taxonomy of psychopathology

## Validation Needed

The 7 Framework makes testable claims:

1. **Fingerprints predict treatment response better than DSM diagnosis**
   - Status: Not yet tested

2. **Change is visible on dimensional scales before categorical remission**
   - Status: Not yet tested

3. **Same "disorder" shows different fingerprints requiring different interventions**
   - Status: Theoretical, supported by comorbidity/heterogeneity literature
