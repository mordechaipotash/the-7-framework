# Interventions: What Moves Each Characteristic

This is where the framework becomes actionable.

**The question**: If someone's GUARD is stuck "up" and they want it "tuned", what actually helps?

## Files

Each characteristic has its own file:

- [head.md](head.md) — Interventions for scattered ←→ locked
- [guard.md](guard.md) — Interventions for down ←→ up
- [heart.md](heart.md) — Interventions for numb ←→ chasing
- [gut.md](gut.md) — Interventions for empty ←→ overflowing
- [grip.md](grip.md) — Interventions for loose ←→ tight
- [ground.md](ground.md) — Interventions for floating ←→ frozen
- [reach.md](reach.md) — Interventions for pulled in ←→ merged

## How to Read

Each intervention specifies:
- **Direction**: Which way does it move the characteristic?
- **Type**: Therapy, medication, lifestyle, somatic, cognitive, behavioral
- **Evidence level**: Research, clinical observation, anecdotal, personal
